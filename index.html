TicTacToeHolbertonStyle

How To

The TicTacToeHolbertonStyle game was developed with Unity using C#.
The code is set up with one class called gameController that holds algorithm.
The algorithm keeps count of the spaces marked.
Please refer to the code snippet below for details on the user flow.

It's intended to be played between two people.
The first player is always the Seahorse and the second player is Holberton.
Seahorse always goes first where it marks a space and then, Holberton.
The winner needs to mark three spaces horizontally, vertically or diagonally.
If it's a draw, the players would need to decide either to restart or rematch.
